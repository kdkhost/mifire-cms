# PWA Icons

Adicione os seguintes ícones neste diretório para o Progressive Web App funcionar corretamente:

## Ícones Obrigatórios

| Arquivo | Dimensão | Formato | Uso |
|---------|----------|---------|-----|
| `icon-192x192.png` | 192×192 px | PNG | Ícone padrão do PWA, tela inicial Android |
| `icon-512x512.png` | 512×512 px | PNG | Splash screen, instalação do PWA |

## Ícones Recomendados (opcionais)

| Arquivo | Dimensão | Formato | Uso |
|---------|----------|---------|-----|
| `icon-72x72.png` | 72×72 px | PNG | Dispositivos de baixa resolução |
| `icon-96x96.png` | 96×96 px | PNG | Favicon alternativo |
| `icon-128x128.png` | 128×128 px | PNG | Chrome Web Store |
| `icon-144x144.png` | 144×144 px | PNG | Windows tiles |
| `icon-152x152.png` | 152×152 px | PNG | Apple touch icon |
| `icon-384x384.png` | 384×384 px | PNG | Splash screen Android |
| `maskable-icon-512x512.png` | 512×512 px | PNG | Ícone maskable (com safe zone) |

## Requisitos

- Use o logo MiFire com fundo vermelho (#DC2626) ou transparente
- Os ícones devem ser quadrados
- Formato PNG com transparência quando apropriado
- O ícone maskable deve ter conteúdo importante dentro da safe zone (80% central)

## Ferramentas Úteis

- [PWA Asset Generator](https://github.com/nicedoc/pwa-asset-generator)
- [Maskable.app](https://maskable.app/) - para testar ícones maskable
- [RealFaviconGenerator](https://realfavicongenerator.net/) - gerador completo de favicons
